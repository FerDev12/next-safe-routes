<Note title='Environment' type='warning'>
  The redirect function created by createNextSafeNavigation can only be used in
  server-components and server-actions. If you require client side navigation
  consider using [useRouter](/docs/beta/navigation/use-router) or the
  [Link](/docs/beta/navigation/link) component.
</Note>
